1. Open Browser:
   → http://localhost:3000

2. Go to Conversations:
   → Click "Conversations" in sidebar
   → OR http://localhost:3000/conversations

3. Click Button:
   → Look for "+ Add Conversation" button (top right)
   → Click it

4. Fill Step 1 (Details):
   Customer: Select from dropdown
   Channel: Select one
   Tone: Select one
   Summary: Type a description
   Click: "Continue → Add Messages"

5. Fill Step 2 (Messages):
   Customer Name: Type
   Customer Message: Type what they said
   Your Name: Type
   Your Message: Type what you said
   Click: "Create Conversation"

6. Watch It Appear:
   ✨ New conversation appears at top of list!
   ✨ Shows customer, channel, tone, summary
   ✨ Data saved to database
